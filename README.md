## ✨ Features

- 🎮 Two Player Mode
- 🔄 Restart Game Option
- 🚫 Detect Win or Draw Automatically
- 🎨 Minimal & Beautiful User Interface
- 🚀 Smooth Gameplay & Fast Performance
- 📱 Responsive for all Android screen sizes
